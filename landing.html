Approfondimento del progetto

Pagina dedicata alla descrizione dettagliata
del progetto GitHub Pages.

Obiettivo

L’obiettivo di questo progetto è dimostrare la capacità di utilizzare Git e GitHub per il versionamento del codice e GitHub Pages per la pubblicazione di un sito web statico.

Git

Utilizzato per il controllo di versione e la gestione delle modifiche al codice sorgente.

GitHub

Piattaforma utilizzata per ospitare il repository e collaborare sul progetto.

GitHub Pages

Servizio che consente la pubblicazione online gratuita del sito web statico.

Risultato finale

Il risultato è un sito web funzionante, accessibile online, composto da più pagine HTML collegate tra loro e versionate tramite Git.
